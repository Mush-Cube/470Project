CookieCoder's Game Dev. Summer 2017 is a 7 day program that will teach youth (ages 12-15) basic software engineering skills to develop games and hopefully after the program, have them pursue a career in software development. Software development is a fascinating field that is in demand in most parts of the world. Major companies like Google and Facebook are looking at the younger generations for a source of innovation to push boundaries in the tech world. Our program will give students an opportunity to get a head start in this competitive world that lives off the creativity of its developer. Our program will start with an introduction to the tools the student will use throughout the program. Tools like sublime text, brackets, android studio, and Visual Studios will be use for creating projects as well as some artistic tools like Photoshop. As the program continues, the students will have daily projects to help them understand how to build games and overall understand concepts in software engineering. At the end of the event, the student will have a solid foundation in which they can pursue thier own interest in this fascinating field.

As you see from the videos, this event is a great learning experience for all. There will be mentors available for use throughout the program's duration. These volunteers are here to answer any technical questions that might arise when doing your projects. These volunteers come from nearby colleges and universities and have extensive knowledge of the material to help out in any situation that might arise.

These are some sample games that will be developed in the program. For demonstrating on this page we used a scripting programing language called JavaScript to make these games. Our students will be making these games from java and C++. As the program goes on they will be making more complicated games that will introduce new concepts and new skills to create them.
